Ниже приведу те команды, которые сам использовал:

$ git init [название проекта] - Создаёт новый локальный репозиторий с заданным именем. Имя задавать не обязательно

$ git clone [url-адрес] - Скачивает репозиторий вместе со всей его историей изменений

$ git status - Перечисляет все новые или изменённые файлы, которые нуждаются в фиксации. А так же показывает в какой мы сейчас ветке

$ git add [файл] - Индексирует указанный файл для последующего коммита.
$ git add * - Индексирует ВСЕ файлы кроме тех, которые начинаются с . точки
$ git add . - Индексирует ВСЕ файлы (вместе с теми файлами, которые начинаются с точки)
